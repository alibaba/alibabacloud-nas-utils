# 阿里云NAS工具集

[📖 English Documentation](./README.md) | 📖 中文文档

alibabacloud-nas-utils项目是阿里云NAS团队开源的一套专业工具集，专门用于处理和分析阿里云文件存储通用型NAS、极速型NAS、CPFS、CPFS智算版的工具集合，帮助用户更好地管理和优化云存储资源。

## 工具列表

### 1. NAS容量统计工具 ([nas_stat_util](nas_stat_util/README.md))
用于分析和统计NAS文件系统的目录级容量使用情况，支持多种筛选条件和输出格式。

## 使用方式

1. 克隆本项目到本地
2. 进入相应工具目录，按使用说明执行所需功能
