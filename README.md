The alibabacloud-nas-utils project is an open-source set of professional tools released by the Alibaba Cloud NAS team, specifically designed for handling and analyzing Alibaba Cloud's General-purpose NAS, Extreme NAS, CPFS, and CPFS for Lingjun, helping users better manage and optimize cloud storage resources.

## Tool List

### 1. NAS Capacity Statistics Tool ([nas_stat_util](nas_stat_util/README.md))
Used to analyze and statistics directory-level capacity usage of NAS filesystems, supporting multiple filtering conditions and output formats.

## Usage

1. Clone this project to your local machine
2. Enter the corresponding util directory and execute the required functions according to the instructions
